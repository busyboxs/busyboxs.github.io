Visual Studio 2019 快捷键

发表于2021-07-25|更新于2021-11-25|VS
|字数总计:640|阅读时长:2分钟|阅读量:
快捷键 描述 说明
Ctrl + G 转到行..
Ctrl + T 转到所有
Ctrl + Shift + T 转到文件
Ctrl + 1, Ctrl + R 转到最近的文件
Ctrl + 1, Ctrl + T 转到类型
Alt + \ 转到成员
Ctrl + 1, Ctrl + S 转到符号
Alt + PgDn 转到文件中的下个问题
Alt + PgUp 转到文件中的上个问题
Ctrl + Shift + BackSpace 转到上一个编辑位置
Ctrl + F 快速查找
Ctrl + H 快速替换
Ctrl + Shift + F 在文件中查找
Ctrl + Shift + H 在文件中替换
Ctrl + Z 撤销
Ctrl + Y 重做
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切
Ctrl + Shift + V 显示粘贴板历史记录
Ctrl + D 复制
Ctrl + K, Ctrl + D 设置文档格式
Ctrl + K, Ctrl + F 设置选定内容格式
Ctrl + Shift + U 转换为大写
Ctrl + U 转换为小写
Alt + 上箭头 将选定行上移
Alt + 下箭头 将选定行下移
Ctrl + K, Ctrl + \ 删除水平空白
Shift + Alt + ] 将选定内容包含到块 选上包含选定内容的块
Shift + Alt + = 展开选定内容 向后扩充选定的内容
Shift + Alt + - 收缩选定内容 向前缩减选定的内容
Ctrl + R, Ctrl + W 查看空白 显示一些空格或者 tab 之类的符号
Ctrl + E, Ctrl + W 自动换行 开启或关闭自动换行
Ctrl + K, / 切换行注释
Ctrl + Shift + / 切换块注释
Shift + Alt + L, Shift + Alt + S 对行排序 对选中的行进行排序
Shift + Alt + L, Shift + Alt + J 联结行 将多行连成一行
Ctrl + K, Ctrl + C 注释选中内容
Ctrl + K, Ctrl + U 取消注释选中内容
Ctrl + J 列出成员
Ctrl + Shift + 空格 参数信息
Ctrl + K, Ctrl + I 快速信息
Ctrl + K, Ctrl + S 外侧代码
Ctrl + K, Ctrl + X 插入代码
Shift + Alt + . 插入下一个匹配的文字光标
Shift + Alt + ; 在所有匹配位置插入文字光标
  • Ctrl + Shift + A : 新建项
  • Ctrl + Shift + N : 新建项目
  • Ctrl + Shift + > : 放大编辑器
  • Ctrl + Shift + < : 缩小编辑器
  • Alt + Shift + arrow : 多选，选择块
  • F12 : 跳转到定义
  • Alt + F12 : 打开定义窗口
  • Esc : 关闭定义窗口
  • Ctrl + Alt + Home : 将定义窗口打开为完整的窗口
  • Shift + Esc : 在定义窗口和原窗口之间切换
  • Alt + ` : 上下文导航菜单
  • Shift + F12 : 查看所有引用
  • F8 : 查找结果或错误列表等的下一个
  • Shift + F8 : 查找结果或错误列表等的上一个
  • Ctrl + Shift + L : 删除行
  • Ctrl + R, Ctrl + O : 重新排序函数参数
  • Ctrl + Alt + L : 打开解决方案窗口
  • Ctrl + Tab : 打开活动窗口
  • Shift + Esc : 关闭当前活动窗口
